## Deskripsi
Aplikasi WebGIS ini dibangun menggunakan **OpenLayers** untuk memvisualisasikan hasil analisis spasial sebaran masjid dan jangkauan layanannya (buffer 500m) di wilayah Kecamatan Siman dan sekitarnya.

## Fitur Utama
* **Peta Interaktif:** Base map menggunakan OpenStreetMap.
* **Analisis Buffer:** Layer visualisasi radius 500 meter dari setiap masjid.
* **Informasi Detail:** Popup interaktif menampilkan nama masjid dan koordinat.
